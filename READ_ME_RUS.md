Данная программа на платформе .NET, реализует игру «Морской бой» в режиме пользователь – компьютер, а также в сетевом режиме пользователь - пользователь. 
Основные правила классического варианта игры: первый игрок «А», расставляет корабли на квадратном игровом поле из n клеток. 
Корабли делятся на классы: одноклеточные, двухклеточные, трехклеточные и четырехклеточные. 
Игрок «В» на своем поле расставляет свои корабли. Корабли не должны касаться друг друга. 
Игра состоит в том, что игроки по очереди называют координаты клеток, в которых, как они предполагают, расположены корабли противника, то есть как бы производят выстрел по выбранной клетке. 
О попадании или промахе игроку сообщается после выстрела. Игра продолжается до тех пор, пока у одного из игроков не будут уничтожены все корабли. 
С целью более удобного проектирования, настройки, и разделения отдельных исполняемых операций, игра «Морской бой» построена на модулях, которые в свою очередь построены на классах взаимодействующих между собой. 
Использование модулей и классов дает улучшенное понимание кода, а значит настройку и последующую модификацию исходного кода программы. 
Данная игра состоит из следующих модулей:
1)	Program – главная точка входа в приложение игры.
2)	Form1 – основная форма приложения игры, вся игра проходит в данной форме.
3)	Chatter – модуль обеспечения игрового чата, лога подсказки и информационной строки.
4)	Connection – форма приложения сетевой игры для создания клиент-серверного подключения.
5)	Server – модуль предназначенный для ПК в роли сервера при сетевой игре.
6)	Client – модуль предназначенный для ПК в роли клиента при сетевой игре.
7)	Settings – модуль заглушка для поддержания работы модуля Settings.Designers нужном при синхронизации сетевой игры. 
8)	AssemblyInfo – модуль параметров сборки и идентификации библиотеки типов СОМ компонентов.
9)	Resources.Designer - создан автоматически классом StronglyTypedResourceBuilder
с помощью Visual Studio.
10)	Settings.Designers - создан автоматически классом StronglyTypedResourceBuilder с помощью Visual Studio, содержит функции синхронизации при сетевой игре.

Инструкция для игры в «Морской бой»
Для игры в морской бой доступно 2 режима: игра с компьютером и сетевая игра с другим игроком, где первый игрок будет в роли сервера, а второй в роли клиента. 
В начале игры, если вы играете по сети с другим игроком:
1)	данный пункт только для игрока, который в роли «Сервер». 
Вам нужно ввести имя сервера и выбрать вариант «Сервер», далее сказать свой IP адрес другому игроку, который в роли «Клиент», для подключения к вам. 
После нужно нажать на кнопку «Подключиться»;
2)	данный пункт только для игрока, который в роли «Клиент». 
Вам нужно ввести имя уже созданного сервера, которое должно полностью совпадать с названием сервера игрока, с которым хотите играть в морской бой. 
После выбрать вариант «Клиент», далее ввести IP адрес игрока, чье имя сервера уже ввели, чтобы подключиться к нему в роли «Клиент». 
Последним действием нужно нажать на кнопку «Подключиться».
Во всем остальном игра «Морской бой» в режиме сетевой игры против другого игрока - полностью повторяет режим морской бой против компьютера. 
В начале игры, если вы играете с компьютером, то сразу выводится приглашение к расстановке кораблей, где расстановка кораблей может быть произведена вручную или случайно. 
Если игрок выполнил недопустимое действие, например, попытался «наложить» корабль на корабль, то ему будет выведено предупреждающее сообщение о его ошибке. 
Для удобства игры добавлено окно чата для игроков по сети, которое каждый ход выводит сообщение о констатации события за ход, окно лога с советами игроку и строка с экспресс информацией 
Если расстановка кораблей закончилась, то игрок нажимает кнопку «Я готов!», которая становится доступна только после расположения всех кораблей, если корабли расставляются вручную. 
Далее выводится приглашение к началу игры.
